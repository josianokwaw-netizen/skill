# skill

Claude Code skill 仓库，存放供 [josianokwaw-netizen/111](https://github.com/josianokwaw-netizen/111) 使用的自定义 skill。

## 目录结构

```
skills/
  <skill-name>/
    SKILL.md
CLAUDE.md       # 仓库行为规范
README.md       # 本文件
```

## 使用方式

每个 skill 位于 `skills/<skill-name>/SKILL.md`，skill 名使用 kebab-case。

详细规范见 [CLAUDE.md](./CLAUDE.md)。
