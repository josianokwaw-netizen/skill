# Skill 仓库 · Claude 行为规范

本仓用于存放 Claude Code 的 skill 文件。

## 目录结构

```
skills/
  <skill-name>/
    SKILL.md    ← skill 定义文件
README.md
CLAUDE.md
```

## Skill 开发规范

- 每个 skill 独立一个子目录：`skills/<name>/`
- 入口文件固定命名为 `SKILL.md`
- skill 名称使用小写字母 + 连字符（kebab-case）
